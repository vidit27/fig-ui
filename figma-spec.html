2
Open Tokens Studio → click Load tokens → paste the contents of figma-tokens.json

Click Apply to document — all colour, spacing and typography variables populate automatically
4
Use Apply to selection on any frame to instantly style it with the AI India token set
5
To change a colour globally, edit the token value once — all components update everywhere

Motion & Micro-interaction Spec
6 interactions. Each designed for low-literacy guidance, action feedback, or cultural delight.
1
Language-First Selector
Trigger: page load · on pill tap
First interaction on the page. Pills display scripts in their own language (हिन्दी not Hindi). Spring-bounce on select, ✓ checkmark prefix appears, toast confirms selection. Guides pre-literate users by making language choice the single entry point.
Easing
cubic-bezier(.34,1.56,.64,1)
Duration
220ms
Scale active
1.08×
Haptic
vibrate([20])
2
CTA Ripple + Pulsing Ring
Trigger: GET STARTED tap
Ripple spreads from touch point (confirms the exact tap registered). Ring pulse radiates outward every 2.5s at idle — draws attention without being distracting. On tap, haptic triple-buzz pattern (20ms–40ms–20ms) mirrors "you did it" confirmation feel from UPI apps.
Ripple duration
550ms linear
Ring pulse period
2500ms ease-in-out ∞

[20, 40, 20]ms
3
Loading Overlay — Diya 🪔
Trigger: after CTA tap
Full-screen navy overlay. Diya emoji flickers (scale + rotate, 1s loop). Progress bar fills green→saffron in 1.8s. 3-step dot tracker advances with spinning active dot, turns green ✓ on complete. Then closes and fires success screen. Familiar flame metaphor = warmth + patience.

4
Diwali Confetti Celebration
Trigger: registration success
60 confetti pieces in saffron / navy / green burst from random positions. Success card springs in with scale(.5,rotate(-5deg)) → scale(1). Strong haptic burst [30,50,30,50,100]ms. Says "you did it!" in a way that transcends literacy — pure visual joy.

Feature Card Shimmer + Icon Bounce
Trigger: hover / tap on feature card
Top accent bar slides in from left (scaleX 0→1). Shimmer sweep crosses the card. Icon scales 1.15× and rotates -5° — "picking it up" micro-gesture. Card lifts -8px with amplified shadow. All feels like touching something real and responsive.

Figma Import Guide
Two ways to get this into Figma — Tokens Studio (recommended) or manual frame setup.
🔌 Method 1 — Tokens Studio Plugin (Recommended)
1
Install Tokens Studio for Figma from the Figma Community (free)
2
Open the plugin → click Load tokens from file
3
Upload figma-tokens.json included in this package
4
Click Apply all tokens to document
5
Colour, spacing, typography styles populate instantly across all frames
🖼️ Method 2 — HTML to Figma Plugin
1
Install HTML to Figma plugin (by Builder.io, free)
2
Open ai-india-fixed.html in Chrome
3
Run the plugin → it screenshots and imports the rendered page as Figma frames
4
Each section imports as a separate auto-layout frame
5
Then apply tokens from figma-tokens.json to make elements editable
📐 Figma Frame Setup (Manual)
1
Create frames: Desktop 1440×auto, Mobile 390×auto. Use Auto Layout, Vertical, gap 0.
2
Create a Colour Styles library: Saffron/500=#FF6B00, Navy/500=#0A1628, Green/500=#0E8C00, plus all tints. Name them brand/saffron, brand/navy, brand/green.
3
Set spacing base = 8 in Layout Grid settings. Use 8/16/24/32/40/48/64/80px for all padding and gap values — never 10, 15, 22, or 25.
4
Create Text Styles: Hero/H1 (Baloo2·800·56), Section/H2 (Baloo2·800·32), Feature/Title (Baloo2·800·18), Body/Base (NotoSans·400·15), Label/XS (NotoSans·800·11·uppercase).
5
Stats: number layer = brand/navy fill. Unit layer (Cr+, .2%) = brand/navy fill. No saffron on stat numbers.
6
Create a Component Set for Feature Card with variants: Default / Hover / Kisan(green) / Health(saffron) / Shiksha(navy) / Sarkari(saffron) / Paisa(green) / Bhasha(navy).
